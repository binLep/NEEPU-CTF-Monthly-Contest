### 来学习如何使用 nc 吧

假如你拿到的域名和端口号如下所示：

`node1.binlep.top:28097`

那么你应该在 linux 下直接使用该命令进行连接：

`nc node1.binlep.top 28097`

或者在 windows 系统上安装完 nc 命令后再使用它

**我们建议使用 nc 命令，以避免一些编码上的错误**

但是在 windows 下，你也可以使用 telnet 命令

命令如下：

`telnet node1.binlep.top 28048`
